About Chinook

Chinook is a sample database containing information that one might find in a music store, with relations such as tracks, artist, genres, and invoices. This web app tech stack includes MariaDB (MySQL server), express for the front end, and a node.js server. This was created to explore the database with a more intuitive UI and as practice working with these technologies. Thanks for visiting!
