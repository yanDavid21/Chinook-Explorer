About Chinook

Chinook is a sample database containing information that one might find in a music store, with relations such as tracks, artist, genres, and invoices. This web app tech stack includes MariaDB (MySQL server), fetchAPI for the front end, and a node.js server. This was created to explore the database with a more intuitive UI and as practice working with these technologies. Thanks for visiting!

The Customer Table

The Customer table represents customers of the store. This table stores the name of a customer, an unique id, company, address,city, state, country, postal code, phone, fax, email, and an employee for each customer. Each customer is helped by an employee and every employee can help several customers so it is a 1 : N relationship, with the Customer table having a foreign key pointing to the employeeID of an employee in the Employee table.
